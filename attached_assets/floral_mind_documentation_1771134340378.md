## 1. Executive Summary

Floral Mind is an AI-native analytics platform that allows non-technical
users to interact with dashboards conversationally, generate deeper
insights, and dynamically create new visualizations without depending on
data analysts.

Instead of static dashboards, Floral Mind introduces an intelligent
analytics agent powered by Claude that understands data context,
generates hypotheses, recommends deeper analysis, and automatically
produces new visualizations.

It transforms dashboards from passive reports into active analytical
partners.

------------------------------------------------------------------------

## 2. Problem Statement

In modern organizations, dashboards are built using tools such as
Tableau and Power BI. However:

-   Client requirements frequently change.
-   Dashboards may not be fully understandable to non-technical users.
-   Clients often want deeper breakdowns after seeing a visualization.
-   Analysts must manually rebuild or modify dashboards.
-   Insight generation is slow and resource-intensive.

Dashboards today are static. Business thinking is dynamic.

------------------------------------------------------------------------

## 3. Solution Overview

Floral Mind is a conversational analytics platform where users can:

-   Upload their dataset
-   Automatically generate dashboards
-   Talk to visualizations
-   Click chart elements and drill deeper
-   Ask natural language questions
-   Generate new charts instantly
-   Receive AI-generated hypotheses
-   Get suggested next analyses

Floral Mind acts as an Autonomous Data Exploration Agent.

------------------------------------------------------------------------

## 4. Core Capabilities

### 4.1 Conversational Dashboard Interaction

Users can type:

-   "Why is Q3 revenue lower?"
-   "Break this down by state."
-   "Show profit margins by category."
-   "Focus only on South region."

The AI: - Understands intent - Generates the required query - Executes
it - Selects appropriate chart type - Returns visual + explanation

------------------------------------------------------------------------

### 4.2 Slice-Based Intelligent Drill Down

When a user clicks a slice in a pie chart or a bar in a graph:

Floral Mind automatically:

-   Generates a hypothesis
-   Suggests possible causes
-   Proposes deeper analysis
-   Creates a new visualization if needed

This creates true agent behavior instead of static filtering.

------------------------------------------------------------------------

### 4.3 Automatic Dashboard Generation

After dataset upload:

Floral Mind: - Detects schema - Identifies numerical and categorical
columns - Generates initial meaningful dashboard - Creates key metrics
automatically

------------------------------------------------------------------------

### 4.4 Structured AI Reasoning

The AI: - Interprets data structure - Plans query logic - Generates
structured output - Validates query format - Executes safely - Produces
visualization + insight

------------------------------------------------------------------------

## 5. Workflow

### Step 1: Data Upload

User uploads CSV file.

### Step 2: Schema Detection

System automatically parses columns and stores dataset in SQL engine.

### Step 3: Initial Dashboard Generation

Floral Mind generates summary metrics and relevant charts.

### Step 4: Conversational Exploration

User interacts via chat or by clicking visual elements. AI generates
SQL, executes queries, and updates dashboard dynamically.
